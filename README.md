# CORE DEBT

Un juego web frenético de supervivencia, gestión de recursos y teclear rápido hecho con HTML5 Canvas y JavaScript puro.

Aquí no vienes a relajarte, vienes a pagar tu deuda.

## ¿De qué va esto?

El juego es básicamente un "boss rush" de gestión. Tienes que farmear recursos contrarreloj perdiéndote por el mapa y entregarlos al Núcleo central. Si eres lento, mueres. Si fallas escribiendo, mueres.

Aquí te explico un poco el proceso de una partida:

### 1. El Comienzo

El menú principal. Directo al grano. Ya te avisa de que esto es "HARD MODE" por defecto. No hay tutoriales fáciles aquí. Dale a START y prepárate.

![Menú Principal](Images/readme1.png)

### 2. El Contexto (Lore)

Un poco de historia para que sepas por qué corres. Año 2402. Una IA divina llamada "THE CORE" nos mantiene vivos, pero a cambio exige recursos constantes. Tú eres un "Debt Collector", un pringado que tiene que currar para pagar la deuda de la humanidad.

![Lore Screen](Images/readme2.png)

### 3. El Gameplay (A farmear)

Este es el bucle principal. Te mueves con el click derecho y farmeas con el izquierdo. Tienes misiones con tiempo límite (arriba a la derecha). Tienes que buscar los materiales específicos que te pide el Núcleo (madera, oro, hierro...) y llevárselos al centro antes de que el contador llegue a cero. Si te lías con el inventario, usa la papelera.

![Gameplay farmeando](Images/readme3.png)

### 4. Los Jefes y Minijuegos

Cada 5 niveles aparece un Boss. No basta con hacerles click. Se activan minijuegos que tienes que superar para romper su escudo. En esta imagen, por ejemplo, toca el modo "AIM MAD", donde tienes que darle a los objetivos amarillos rápidamente. Otros bosses te harán escribir palabras a toda velocidad.

![Minion Boss Aim Game](Images/readme4.png)

### 5. El Final Boss (Nivel 67)

Si sobrevives a la locura hasta el nivel 67, te enfrentas a esto: "GOD". Es el Núcleo cabreado. Tiene una barbaridad de vida y los minijuegos de escribir aquí no perdonan fallos. Si consigues bajarle la vida a cero, habrás pagado la deuda (por ahora).

![Final Boss God](Images/readme5.png)

* HTML5 Canvas puro (sin librerías gráficas).
* Vanilla JavaScript.
* CSS para la UI.
* Delta Time implementado para que funcione igual en monitores de 60Hz y 144Hz.
